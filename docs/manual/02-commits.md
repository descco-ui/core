## Commit Message Regex

`(feature|hotfix|bugfix|fix|docs|style|refactor|test|chore)\(.*\)\:.*\[BCSPACEY\-.*\]`

## Prefix

- feature
- hotfix
- bugfix
- fix
- docs
- style
- refactor
- test
- chore

## Slug

Between `()` and add `:`

## Message

Lowercase letters

## Taks number

Between `[]`

## Example

feature(storybook): organization of items [BCSPACEY-XXX]
